# Life After Reset

AI-powered life simulation game built with Python, Streamlit, and Ollama.

## Overview
You play as someone who wakes up from a 15-year coma and must rebuild their life in the city of Las Playas. Guided by Tom, a slightly sarcastic but supportive life coach, every decision you make shapes your future.

## Features
- Choice-based gameplay (4 options every turn)
- Hidden law vs crime system
- Police heat and consequences
- Inventory and survival mechanics
- Persistent memory and storytelling
- Terminal-style UI

## Tech Stack
- Python
- Streamlit
- Ollama (local LLM)
